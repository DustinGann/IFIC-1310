Graceful Degeneration and Progressive Enhancement

Graceful degradation seems to be a mindset where the programmer focuses on making a website optimized for the biggest baddest browsers and just keeping it functional on any subpar browsers. Progressive enhancement is where you instead focus on making sure your content is as accessible and digestible as possible. Progressive enhancement is the better way of going about things because ultimately, the content is the whole point of why you build a website. If someone can't comfortably access your content they will just move on. So you have to create a website that is usable even if some of the code isn't compatible or working. If this is done it will be more available for more people to access.
